# 分享一款简单粗暴的社工库源码

## 杂谈

之前出过专门一期社工库搭建的视频教程

很多人反馈如果自己有txt、csv的数据，又不会搭建数据库，怎么办？今天给大家分享一个简单粗暴的社工库源码

## 声明

本教程仅用于教育目的，请勿用于非法用途。

## 介绍

你只需要把你的txt文件、csv文件放到`FancyPig`目录下

搜索功能就可以自动从`FancyPig`目录下进行模糊查询了，如果文件太大了或者比较多可能查询会比较慢

## 环境需要

需要使用LNMP或者LAMP的环境
要有php运行环境就行
